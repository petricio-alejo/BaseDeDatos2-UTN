# Base de Datos II - UTN

Repositorio de trabajos prácticos de la materia **Base de Datos II** de la Universidad Tecnológica Nacional (UTN).

## Contenido

- **TP1**: Fundamentos, Integridad y Concurrencia
- **TP2**: MongoDB - CRUD, Aggregation y operaciones avanzadas
- **TP3**: Pipeline de Agregación en MongoDB

## Objetivo

Este repositorio refleja los conocimientos adquiridos durante la cursada, incluyendo:
- Integridad y gestión de transacciones
- Optimización de consultas e índices
- Bases de datos NoSQL con MongoDB
- Agregaciones y pipelines de procesamiento
- Seguridad y administración de bases de datos
